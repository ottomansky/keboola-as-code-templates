The `Shoptet Ads & Sales Performance Orchestration` flow is designed to streamline the extraction, transformation, and application of data related to ads and sales performance. It consists of three sequential phases, each building upon the previous to ensure a comprehensive data processing pipeline.

## Step 1
This phase focuses on data extraction from various sources. The tasks in this phase extract data from Google Analytics, Facebook Ads, and Shoptet Permalink. These components gather raw data necessary for subsequent processing and analysis.

## Step 2
The second phase is dedicated to data transformation. The task in this phase utilizes Python to transform the extracted data, preparing it for application use. This step ensures the data is in a suitable format for further analysis and reporting.

## Step 3
The final phase involves applying the transformed data. The task in this phase uses data applications to leverage the processed data for insights and decision-making. This step completes the flow by enabling actionable outcomes based on the refined data.
